# Setup

```
git clone https://github.com/BenjaminHabert/inria-aphp.git
cd inria-aphp

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# running notebooks
export PYTHONPATH=$PYTHONPATH:$(pwd)
jupyter notebook
```
